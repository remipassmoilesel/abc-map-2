## Launch demo

	$ git clone https://github.com/remipassmoilesel/abcmap2
	$ cd abcmap2
	$ git submodule init
	$ git submodule update
	$ cd lib/imagej-surf/
	$ mvn install
	$ cd ../geotools/modules/plugin/imagemosaic-jdbc
	$ mvn install
	$ cd ../..
	$ maven compile
	$ maven exec:exec
